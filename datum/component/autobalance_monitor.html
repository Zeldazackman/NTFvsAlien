autobalance_monitor
/datum/component/autobalance_monitor

A component that actively sets the value of requisition points for this item in /datum/controller/subsystem/monitor. Points are changed whenever the gamestate changes to shipside OR the item changes z-level. Should be applied to meaningful requisition items.
